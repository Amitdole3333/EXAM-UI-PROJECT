# Eduport Education Theme UI

This project is a static front-end UI for an online learning and teaching marketplace, inspired by the "Eduport" theme. 

## Overview
The page features a responsive layout built using HTML5 and CSS with the help of **Bootstrap 5**. It showcases a modern e-learning platform interface including navigation, category dropdowns, course lists, and promotional banners.

## Features
- **Responsive Navigation Bar**: Includes a logo, category select dropdowns, search bar, and user profile icon.
- **Hero Section**: A promotional area with a call-to-action to get started with learning.
- **Popular Courses Grid**: A display of multiple course cards with images, titles, ratings, and duration.
- **Footer**: A comprehensive footer section containing company links, teaching guidelines, and contact information.
- **Icons & Styling**: Uses Bootstrap Icons for visual elements and custom CSS for layout adjustments.

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5.3.3
- Bootstrap Icons

## Setup Instructions
1. Clone or download the repository.
2. Open the `index.html` file in any modern web browser to view the application. No server or build process is required since it's a static HTML file.

## Project Structure
- `index.html`: The main HTML file containing the structure of the webpage.
- `public/images/`: Directory containing static images used throughout the page.
